\# 💰 Personal Expense Tracker



A simple, interactive expense tracking web app built with Python, Streamlit, SQLite, and Plotly. Designed to help users log daily expenses, visualize spending patterns, and stay within budget.



\## Features



\- \*\*Add \& View Expenses\*\* — Log expenses with date, category, description, and amount

\- \*\*Interactive Charts\*\* — Visualize spending trends and category-wise breakdowns using Plotly

\- \*\*Budget Alerts\*\* — Get notified when spending approaches or exceeds set budget limits

\- \*\*Persistent Storage\*\* — All data stored locally using SQLite



\## Tech Stack



\- \*\*Frontend/UI:\*\* Streamlit

\- \*\*Database:\*\* SQLite

\- \*\*Visualization:\*\* Plotly

\- \*\*Language:\*\* Python

\## Getting Started



\### Prerequisites

\- Python 3.8+



\### Installation



1\. Clone the repository

```bash

git clone https://github.com/divyayata1623/expenses-tracker.git

cd expenses-tracker

```



2\. Install dependencies

```bash

pip install -r requirements.txt

```



3\. Run the app

```bash

streamlit run app.py

```



\## Sample Data



This repo includes `sample\_data/expenses\_sample.db` with dummy expense entries so you can explore the app's features without needing your own data.
